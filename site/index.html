Painel Cheat

Baixe o aplicativo desktop que reúne o painel web com backend local.

Download Linux AppImage Download Linux DEB

Como instalar

  1. Execute npm install na raiz do projeto.
  2. Use npm run dist para gerar os instaladores em dist/.
  3. Coloque os arquivos gerados em dist/ e retorne a esta página para baixar.

O aplicativo inicia o servidor backend local e carrega o painel Web automaticamente.
