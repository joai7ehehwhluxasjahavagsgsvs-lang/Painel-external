Painel Cheat

Baixe o aplicativo desktop e aproveite o painel web com backend local em um só pacote.

Download Linux AppImage Download Linux DEB

Instaladores disponíveis

  • Linux AppImage — arquivo portátil para Linux.
  • Linux DEB — instalador Debian/Ubuntu.

Se você ainda não gerou os instaladores, acesse a raiz do projeto e rode npm run dist.

Como usar

  1. Baixe o instalador desejado.
  2. Abra o arquivo no seu sistema e instale o app.
  3. Execute o aplicativo para iniciar o servidor backend local automaticamente.

O app carrega o painel web local e mantém a interface disponível em modo desktop.

App Android

O aplicativo Android está disponível no código-fonte em cheat-panel/android/. Para gerar o APK, use:

cd cheat-panel/android
./gradlew assembleDebug

O APK será gerado em cheat-panel/android/app/build/outputs/apk/debug/app-debug.apk.
